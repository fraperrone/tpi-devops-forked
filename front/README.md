# Frontend: To‑Do List

Aplicación frontend simple (sin backend) para gestionar una lista de tareas.

Características:
- Añadir, editar (doble clic o Enter), eliminar tareas.
- Marcar tareas como completadas.
- Filtrar (Todas / Activas / Completadas).
- Borrado de completadas.
- Persistencia usando localStorage (clave: `tpi_todos_v1`).
- Accesible y responsive.

Cómo usar:
1. Abre `front/index.html` en tu navegador (doble clic o desde el explorador de archivos).
2. Escribe una tarea y pulsa "Añadir" o Enter.
3. Haz doble clic sobre el título de una tarea para editar.

Notas para desarrollo:
- Archivos principales: `index.html`, `style.css`, `app.js`.
- No requiere instalación de dependencias.
